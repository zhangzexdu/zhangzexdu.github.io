    函数定义的几种方式

三种方式比较

- function语句方式优先解析：javascript解释器会优先解析。
- function方式和直接量方式是静态的，编译一次，放置在内存中，执行效率高，而构造方式虽然不占用内存，但是需要每次调用构造函数，效率比较低，构造函数方式是动态的
- 直接量方式和function方式效率比较高，而构造函数方式执行效率慢
