毕业了

古都徘徊


2013年，我毕业了，带着满腔的兴奋与挣钱的渴望留在了古都——西安：一座有着及其历史韵味的现代化城市，虽然自己一直在这座城市生活了四年，但毕业后才发现古都西安与我先前认识的不一样，每天早晨醒来打开窗清新的空气，但到了下午下班回家的时候，走在路上呼吸着污浊的空气，灼热的地面让人不由得厌恶这个城市。我住的地方离上班的地方也不是太远，每天上下班穿梭在密流的人群中，眼瞅着各色美眉，都是在高新区干着劳动密集型的年轻俊男靓妹，还有一部分是工作好几年的，而我，只是一个刚进入社会的毛头小子。
上班没一个月，就被安排出差，刚开始有种兴奋，到了之后才发现所有的都与自己想象的不一样，才发现出差就意味着每天晚上加班，周末加班，没有假期，没有休闲时间，看着老大哥们一个个精神抖擞的每天高谈阔论的嘻嘻哈哈，不知道为什么我总是感觉心情不好，就好像是在混日子，打发光阴，唯有的兴奋变成了每个月发工资的时候，能让我们这些小伙伴们小小的喜悦一下，仅此而已。

背起行囊,上车走了

柳暗花明

2016，太多太多

我来了，八月
